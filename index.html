Learn to Code - Recording Audio

Just doing it is a way to learn. This excercise starts with code and stays there, presenting a webapp for recording voice, adding additional features like cloud API's in subsequent lessons. Simple javascript code using an interactive approach: running, editing, and rerunning code is a great way to learn. In lesson # 1, See how less than 15 lines of javascript provide an audio recorder. Understand the foundation of a small js app for media recording. Follow links if you want to drill down further.
